* 加總 : $$S_x value = \sum^{N}_{i=1}{d_{xi}}$$

```text
# _ : 下標
# ^ : 上標
# \sum : 加總符號
# {  } : 一個項目
S_x value = \sum^{N}_{i=1}{d_{xi}}
```

* 分數 : $$\beta_{ij} = d_{ij}-\frac{S_i + S_j}{N-2}$$

```text
# \frac{A}{B} : 分數, A 為分子，B 為分母
\beta_{ij} = d_{ij}-\frac{S_i + S_j}{N-2}
```
