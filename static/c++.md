| -- | -- | -- |

| long labs(long); | long res = labs(-2); //res = 2 | cstdlib |

| double log(double); | 返回 $$log_{e}$$ 的值| cmath |
| double log10(double); | 返回 $$log_{10}$$ 的值 | cmath |
