###Background
---

* JQuery 是一套 Javascript 使用的 library。JQuery 亦有額外的擴充元件，可以參考 JQuery UI。

* JQuery 主要用於 Document Object Model (DOM) 文件的操作 (參考 HTML DOM 快速導覽)，針對物件的操作，如
  1. 文件物件、
  2. 元素物件、
  3. 事件物件、
  4. html 元素事件處理、
  5. windows 物件、
  6. 樣式設定物件、
  7. 樣式設定物件 style、
  8. html 元素物件等。

* 舉例而言，可以快速選取多個物件，並針對這些物件進行處理，並且加強了 Ajax (非同步傳輸)與事件 (Event) 的功能。
